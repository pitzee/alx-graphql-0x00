# Character Info by ID - Rick and Morty GraphQL API

This project contains GraphQL queries to fetch specific character details from the Rick and Morty API.

**Endpoint:** https://rickandmortyapi.com/graphql

**Query Fields:**

- id
- name
- status
- species
- type
- gender
